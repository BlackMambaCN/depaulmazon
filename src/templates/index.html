This application serves as a DePaul CDM course and instructor recommender, the recommendations are made based on the user profile, which includes your major, degree level, concentration and interests, more specific profiling options might be implemented later on.

To use this application, you just need to select different options based on your personal profile and preferences, then the system will return some classes or teachers you might be interested, and the links to official school pages of those courses or instructors will be provided.

If you run across any problem or question you can contact me through my email, but immediate responses are NOT guaranteed :)
